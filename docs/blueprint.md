# **App Name**: SyntheticPei

## Core Features:

- User Authentication: Enable users to create accounts, log in, and manage their profiles.
- Court Availability: Display available synthetic courts in Pereira, including location and pricing.
- Online Booking: Allow users to book courts online for specific dates and times.
- Payment Processing: Integrate secure online payment processing for booking reservations.
- Booking Management: Enable users to view, modify, or cancel their existing reservations.
- Admin Dashboard: Provide administrators with a dashboard to manage courts, bookings, and user accounts.
- Payment Tracking: Enable admin users to track the payment status of bookings (pending, realized).

## Style Guidelines:

- Primary color: Vibrant green (#4CAF50) to evoke the lushness of synthetic turf.
- Background color: Light green (#E8F5E9), a heavily desaturated shade of the primary.
- Accent color: Yellow-green (#8BC34A), a harmonious analogous shade of the primary that pops a bit.
- Font pairing: 'Poppins' (sans-serif) for headings, 'PT Sans' (sans-serif) for body.
- Use sports-themed icons to represent various court types and booking actions.
- Implement a clean and intuitive layout with clear sections for court listings, booking calendar, and user profiles.
- Incorporate subtle transitions and animations to enhance user interactions, such as loading indicators and confirmation messages.
